## Instructions for use 

The code accepts the following flags:  

* --micro 'type of microscrope' (Default: Arctica)  
* --d 'diameter of particle' (Default: 150 Angstroms)  

__Example__  

```bash
python BoxSizePrediction.py --micro Arctica --d 55
```
